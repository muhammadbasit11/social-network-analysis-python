# Social Network Analysis with Pure Python

This project simulates core social network features using a JSON dataset and Python.

## Key Components

- Data loading & exploration
- Data cleaning & validation
- Friend recommendations (People You May Know)
- Interest-based recommendations (Pages You Might Like)

## Techniques Used

- Dictionaries & sets for relationships
- Deduplication & filtering logic
- Mutual connection analysis
- Basic collaborative filtering

## Notes

This project intentionally avoids Pandas/NumPy to strengthen core Python skills.
